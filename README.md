# Citizen Reporting Mobile App

## Overview
This is a mobile application built using Apache Cordova, HTML, CSS, and JavaScript. The app allows users to report incidents such as accidents, fighting, and rioting.

## Features
- Add new incidents
- View all incidents
- Filter incidents by category
- Capture geolocation (latitude & longitude)
- Upload images
- User login system
- View personal incidents

## Technologies Used
- HTML
- CSS
- JavaScript
- Apache Cordova

## Installation
1. Clone the repository
2. Run the project using Cordova:
   cordova run browser

## APK
The app was built into an Android APK and tested on a physical device.
